# CodeAlpha Language Translation Tool

An AI-powered, full-screen language translator built with Python. This project was developed as part of the **CodeAlpha Artificial Intelligence Internship**.

## 🚀 Features
- **Modern UI:** Built with CustomTkinter for a sleek dark-mode aesthetic.
- **Real-time Translation:** Uses Google Translate API.
- **Full-Screen Design:** Optimized for high visibility and professional use.
- **Multi-language Support:** Supports over 100 languages including Tamil, Spanish, French, etc.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** CustomTkinter, Googletrans API
